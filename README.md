# Bank Fraud Detection App

This project is a simple fraud detection system built using machine learning (Isolation Forest).  
The app takes transaction details as input and predicts whether the transaction is **fraudulent or normal**.

### Features
- Detects unusual or risky transactions  
- Accepts user input through a Streamlit interface  
- Displays prediction results clearly  
- Model saved with joblib and deployed to Streamlit Cloud  

### Technologies Used
- Python  
- Isolation Forest  
- Streamlit  
- Pandas, NumPy  
- Scikit-learn  

### Deployment
The app is deployed on **Streamlit Cloud** and the source code is available in this GitHub repository.
